---

## ✨ Características / Highlights

### ✅ Arquitectura limpia (Clean Architecture)
- Separación clara de **capas**: *data*, *domain* y *presentation*.
- **Principios SOLID** aplicados para facilitar testeo y evolución.
- Dependencias dirigidas hacia el dominio.

### ⚡ Jetpack Compose
- UI moderna declarativa, rápida y escalable.
- Reutilización de componentes y estado controlado.

### 🔌 Inyección de dependencias con Koin
- Configuración simple y legible.
- Módulos desacoplados y fáciles de extender.

### 🌐 API con Ktor
- Cliente HTTP ligero y moderno.
- Manejo explícito de errores y respuesta tipada.

## 🧭 Estructura del proyecto / Project Structure

